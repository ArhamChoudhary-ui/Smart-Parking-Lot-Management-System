# Desktop Application (Qt)

This folder contains the Qt/C++ desktop application.

## Layout

- `src/`: C++ source files (`.cpp`)
- `include/`: Header files (`.h`)
- `resources/`: Qt resource files (`.qrc`)
- `ParkingGUI.pro`: qmake project file

## Build

Open `ParkingGUI.pro` in Qt Creator and build using your Qt kit.
